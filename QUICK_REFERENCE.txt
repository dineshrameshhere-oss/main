## Entry Rules
```
✅ ENTER when ALL true:
├─ RSI > 70 (strong BUY) OR RSI < 30 (strong SELL)
├─ Volume > 1.5× average (confirmation)
├─ EMA3 > EMA8 (for LONG) or EMA3 < EMA8 (for SHORT)
├─ NOT 9:15-9:30 (skip opening chaos)
├─ NOT 12:00-13:15 (skip lunch lull)
├─ NOT 14:45-15:30 (skip end-of-day theta)
└─ Strike = ATM ± 150 points OTM (delta ≈ 0.35)
```

## Exit Rules
```
❌ EXIT when ANY triggers:
├─ TP HIT: Premium +25% (main goal)
├─ SL HIT: Premium -5% (quick loss)
├─ TIME: 45 min hold max (avoid overnight theta)
└─ EOD: 15:15 IST mandatory close
```
